# Badge Project
A personal project for a custom animated convention badge running on ESP32-S3 microcontroller

## Project structure
- #### badge-project-cpp
  - Firmware for the microcontroller, written primarily in C++
- #### badge-project-android
  - Android app to act as a remote control for the device, written in Kotlin
- #### badge-project-tools
  - CLI tools for image and data conversion to create files the firmwware can read, written in Rust
